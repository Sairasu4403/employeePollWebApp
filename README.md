# employeePollWebApp - Udacity Project

A polling web application where authenticated users can create “Would You Rather” style questions, vote on polls, view results, and check the leaderboard.  
This project is created for a Udacity project submission.

---

## Features

- **Authentication** (login / protected pages)
- **Dashboard** showing answered and unanswered questions
- **Poll Details** with voting and results view
- **Create New Question** (two options)
- **Leaderboard** ranking users by activity
- **404 Not Found** route for invalid pages
- **Automated Tests** (unit/component tests included)
